# Проект: Статистический учет пациентов стационара

## Описание
Данный проект предназначен для ведения статистического учета пациентов стационара.  
Он позволяет:

- Хранить данные о пациентах ФИО, пол, год рождения, дату поступления.
- Вести учет диагнозов и истории заболеваний.
- Формировать отчеты по диагнозам с деталями пациентов.
- Управлять пользователями системы администраторы, врачи, медсестры.

## Технологии
- PHP (для серверной логики)
- MySQL (база данных)
- HTML/CSS (интерфейс фильтров и отчетов)
- PDO (для работы с базой данных)

## Структура проекта
- `/config/` — настройки базы данных
- `/scripts/` — скрипты миграции и наполнения базы (`migrate.php`, `seed.sql`)
- `/public/` — файлы веб-интерфейса
- `/SQL/` — SQL-файлы с таблицами и начальными данными
- `/app/` — основной PHP-код приложения



## Запуск проекта

1. **Настройка базы данных**  
   - Создайте базу данных в MySQL, например `hospital_stat`.
   - Откройте файл `.env` и укажите:
     - имя базы (`dbname`)
     - пользователя и пароль (`user` и `pass`)
     - хост (`host`)

2. **Миграция базы данных**  
   - Откройте терминал в папке проекта (`\hospital-stat\`)  
   - Выполните команду:
     ```bash
     php scripts/migrate.php
     ```
   - Скрипт создаст таблицы из `SQL/migrate.sql` и наполнит их тестовыми данными из `SQL/seed.sql`.

3. **Запуск веб-интерфейса**  
``` http://localhost/hospital-stat/```



## Цель
Проект служит инструментом для анализа и статистики пациентов стационара, облегчая работу медицинского персонала и администрации больницы.
